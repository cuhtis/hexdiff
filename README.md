# Hexdiff

Get the diff of binary files.

Implemented by reversing the solution to the Greatest-Common-Subsequence problem for two byte streams.

Created by Curtis Li.

## Usage

```
./hexdiff FILE1 FILE2
```

## Installation

```
git clone https://github.com/curtis99877/hexdiff.git
cd hexdiff && make
```

## Features

* Get the diff of binary files
